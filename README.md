# KanMind Backend

## Beschreibung
Dieses Projekt ist ein Backend für eine Kanban-App, entwickelt mit Django und Django REST Framework.

Es stellt eine REST-API bereit, über die Boards, Tasks und Kommentare verwaltet werden können.
